# Shipping Management System (C++)

## Description
This project is a console-based Shipping Management System developed using Object-Oriented Programming concepts in C++.

## OOPS Concepts Used
- Inheritance
- Polymorphism
- Encapsulation
- Virtual functions

## Package Types
- Letter
- Box
- Fragile

## How to Run
1. Compile using any C++ compiler
2. Run `main.cpp`
